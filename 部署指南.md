# 🌐 部署到 Streamlit Cloud 指南

> 让其他人通过公网链接访问你的财政政策舆情监测系统

---

## 前提

- 一个 **GitHub** 账号（ https://github.com/signup ）
- 一个 **Streamlit Cloud** 账号（用 GitHub 登录 https://streamlit.io/cloud ）

---

## 第一步：安装 Git 并推送代码

### 1.1 下载安装 Git

浏览器打开 https://git-scm.com/download/win → 下载 → 一路默认安装

### 1.2 配置 Git

```powershell
# 打开 PowerShell 或 CMD，执行：
cd C:\Users\jjt\Desktop\财政政策舆情监测
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"
```

### 1.3 初始化 Git 仓库

```powershell
git init
git add .
git commit -m "初始提交：财政政策舆情监测系统"
```

### 1.4 在 GitHub 创建仓库

1. 浏览器打开 https://github.com/new
2. 仓库名填：`fiscal-policy-monitor`
3. 选 **Public**（免费）
4. 不要勾选任何初始化选项
5. 点击 **Create repository**

### 1.5 推送代码到 GitHub

```powershell
git remote add origin https://github.com/你的用户名/fiscal-policy-monitor.git
git branch -M main
git push -u origin main
```

---

## 第二步：部署到 Streamlit Cloud

### 2.1 登录并创建 App

1. 打开 https://streamlit.io/cloud
2. 点击 **Sign in with GitHub**
3. 点击 **New app**
4. 选择刚创建的 `fiscal-policy-monitor` 仓库
5. Branch 选 `main`
6. Main file 填 `streamlit_app.py`

### 2.2 配置密钥（重要！）

点击 **Advanced settings** → **Secrets** → 填入：

```
DEEPSEEK_API_KEY = sk-210eb0617fdf4cc38029aa4c664f15e3
FONT_PATH = /usr/share/fonts/truetype/wqy/wqy-microhei.ttc
```

> `FONT_PATH` 指向 Linux 服务器上已安装的中文字体，`packages.txt` 会自动安装

### 2.3 点击 Deploy

等待 3-5 分钟构建完成，你会得到一个链接：
```
https://你的用户名-fiscal-policy-monitor.streamlit.app/
```

把这个链接发给任何人，他们就能访问了！

## 国内访问慢的替代方案

如果 GitHub / Streamlit Cloud 在国内访问慢，可以用 **Hugging Face Spaces**：

1. 打开 https://huggingface.co/spaces
2. 点击 **Create new Space**
3. Space name 填 `fiscal-policy-monitor`
4. SDK 选 **Streamlit**
5. 将 `requirements.txt` 拖入文件上传区
6. 通过 web 界面上传代码文件（或连接 GitHub）
